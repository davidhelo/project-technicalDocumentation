CSS introduction

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

CSS is designed to enable the separation of presentation and content, including layout, colors, and fonts. This separation can improve content accessibility, provide more flexibility and control in the specification of presentation characteristics, enable multiple web pages to share formatting by specifying the relevant CSS in a separate .css file which reduces complexity and repetition in the structural content as well as enabling the .css file to be cached to improve the page load speed between the pages that share the file and its formatting.

Separation of formatting and content also makes it feasible to present the same markup page in different styles for different rendering methods, such as on-screen, in print, by voice (via speech-based browser or screen reader), and on Braille-based tactile devices. CSS also has rules for alternate formatting if the content is accessed on a mobile device.

Definitions

In CSS there are three types of definitions:

  • Inline:

    This is when the styling is defined in the HTML element tag. For Example:

    <p style="color: red;"> This paragraph will be red. </p>

    Result:

    This paragraph is be red.

  • Internal:

    For this, the CSS formating is defined in the HTML tag <style> inside the HTML document. For Example:

    <style> p { color: blue; } </style>

    Result:

    This paragraph is be blue.

  • External:

    This type of declaration is made in a separated CSS document. For Example:

    p { background: green; }

    Result:

    This paragraph has green background.

Selectors

The selectiom determines which part of the markup the style is applied. The most important selectors are:

  • Type selector:

    This selects all the elements of a specific type. For Example, the next code will apply blue color to all the h1 elements in the HTML:

    h1 { color: blue; }
  • Id selector:

    This selects all the elements of a specific id attribute. For Example, the next code will apply blue color to all elements with id="hello", for id's the character # should be used before:

    #hello { color: blue; }
  • Class selector:

    This selects all the elements of a specific class. For Example, the next code will apply blue color to all elements with class="bye", for class selectors should be used dot character before:

    .bye { color: blue; }
Descendant selectors

This selector consists in defining elements that are parented. For example, the next code will apply the style to all the <p> elements that are inside elements with "#container" id:

#container p { color: blue; }

Adding a > character will restrict in a way that strictly only the <p> elements which are directly child of "#container" id's. Example:

#container > p { color: blue; }
